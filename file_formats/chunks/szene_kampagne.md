# Chunk SZENE_KAMPAGNE #

Always 4 bytes. Contains a 32bit (unsigned??) integer.
Use unknown.
If present, it's the first **Chunk**.
Files containing this **Chunk** have an number at the end before the file ending.
eg: `Some Scenario0.szs`

## Observed Values ##

- 0
- 1
- 2
- 3
- 4
- 5
