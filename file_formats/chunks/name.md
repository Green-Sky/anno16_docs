# Chunk NAME #

This [Chunk](../chunks.md) contains the **Name** of the Scenario/Savegame.

## Memory Layout ##

After the [Chunk Header](../chunks.md#chunk-header) is a C-String (not '\0' terminated??) of the length of the **Chunk**.
